WP Hotkeys provides time-saving keyboard shortcuts to help you quickly navigate the WordPress dashboard.

== Description ==
WP Hotkeys helps you navigate your dashboard as quickly as possible with fully customizable keyboard shortcuts.

= Settings =
WP Hotkeys settings can be found at *Settings > WP Hotkeys*.

= Features =
* Works right out of the box with built-in default Keyboard Shortcuts for each dashboard menu item.
* Hotkey hints next to each menu item help you remember your shortcuts (can be turned off).
* Fully customizable - define your own Keyboard Shortcuts.
* Lightweight - less than 2kb minified jQuery.
* Built-in warning to let you know if you have duplicate Keyboard Shortcuts.

= Coming Soon... =
* Export/import your favorite hotkey setup

== Installation ==
1. Install the plugin.
2. You\\\'ll now see hotkey hints displaying next to your dashboard menu items.
3. Go to *Settings > WP Hotkeys* to create your own custom Keyboard Shortcuts, toggle hotkey hinting on/off, and more.
